Веб-додаток з ім'ям Learning Log, за допомогою якого користувач зможе вести журнал
тем, що його цікавлять, і створювати записи в журналі під час вивчення кожної теми. 
Домашня сторінка Learning Log містить опис сайту та запрошує користувача зареєструватися 
або ввести свої облікові дані. Після успішного входу користувач отримує можливість 
створювати нові теми, додавати нові записи, читати та редагувати наявні записи.

Мої команди для запуску
cd D:\Python39\web_app\blog
python manage.py runserver

python -m venv ll_env
ll_env\Scripts\activate
